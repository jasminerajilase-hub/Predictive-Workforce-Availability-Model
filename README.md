# Predictive Workforce Availability Model

## Project Overview

This project predicts workforce availability using machine learning techniques and visualizes workforce analytics through an interactive Power BI dashboard.

The solution helps operations managers proactively identify staffing risks, overtime issues, fatigue trends, and employee availability patterns.

---

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* XGBoost
* Random Forest
* SQLite / SQL
* Power BI

---

## Machine Learning Workflow

### Data Processing

* Cleaned workforce employee dataset
* Handled missing values
* Performed feature engineering
* Created fatigue and seasonal trend features

### Models Used

* Random Forest Classifier
* XGBoost Classifier

### Best Model Performance

* XGBoost Accuracy: 96%

## Dashboard Features

* Workforce availability monitoring
* Department-level availability analysis
* Overtime vs availability analysis
* Fatigue risk tracking
* Shift performance monitoring
* Employee risk indicators
* Workforce trend forecasting
